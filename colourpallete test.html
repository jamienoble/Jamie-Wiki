How to Use This Palette
  • Group 1: Use for backgrounds in dark mode, or text in light mode.
  • Group 2: Use for backgrounds in light mode, or text in dark mode.
  • Group 3: Use for UI highlights, selection, code syntax, links, etc.
  • Group 4: Use for warnings, errors, success, or colorful accents.
Palette Usage Guide
  • Avoid brown on green, or green on brown for backgrounds and menus.
  • If background is Forest Night, use Moss for menus or overlays.
  • If background is Soil, use Charcoal for menus or overlays.
  • For text on dark backgrounds, prefer light colors from Sunlit Earth.
  • For text on light backgrounds, prefer dark colors from Deep Earth.
